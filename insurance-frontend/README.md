# Insurance Frontend (React)

Professional React frontend for the Insurance Policy & Claim Management System.

## Tech Stack

- React 18
- Vite
- React Router v6
- Axios
- Custom modern CSS layout

## Getting Started

```bash
npm install
npm run dev
```

By default the app expects the backend at `http://localhost:8080`.
You can override this by creating a `.env` file with:

```bash
VITE_API_BASE_URL=http://localhost:8080
```

Then visit `http://localhost:5173` in your browser.
